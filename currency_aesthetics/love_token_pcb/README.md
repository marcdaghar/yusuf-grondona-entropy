# PCB Love Token — Carte Électronique Symbolique

Ce dossier contient un design de PCB (circuit imprimé) qui n'a **aucune fonction électronique**. C'est un objet transitionnel.

## Forme
- Cœur aux proportions Φ (1:1.618)
- Pistes en spirale de Fibonacci
- Trou central pour suspendre (comme un billet plié)

## Signification
- Il ressemble à une carte de crédit → active l'attente d'une transaction
- Mais il ne sert à rien → bloque le réflexe marchand
- Il est fait pour être porté → devient talisman

## Fichiers
- `love_token.kicad_pcb` — pour KiCad (logiciel de CAO électronique)
- `love_token.svg` — pour visualisation directe

## Fabrication
Commander sur JLCPCB ou PCBWay :
- Format : 54mm × 86mm (carte de crédit)
- Épaisseur : 1.6mm
- Finition : OR ENTER (gold finish) — Φ oblige
